Detailed Description

template<typename T>
class mymatrix::Matrix< T >

A class that represents a mathematical object written in the form of a rectangular table of elements.

See Matrix(mathematics)

The following operations are implemented:

  • adding and subtracting matrices;
  • increase and decrease of the matrix size;
  • creating an empty matrix;
  • etc.

The following operations must be overrided for items:

  • operator+ ;
  • operator- ;
  • operator<< .
